What Pastemark is?

Pastemark is one of two-evening projects, which in designed to solve one problem, but solve it really well.

No, I asked what Pastemark really is?

Well, it's easier to explain who needs Pastemark. They are people who have to type similar texts (posts, mails, etc.) every day and don't want keep repeating themselfs. Pastemark will create easy-to-use bookmarklet for them which will paste text snippet into field they want.

Why Pastemark is better than .txt file and Notepad?

Of course, you can keep your text snippets in .txt file and copy-paste it from Notepad, if you like. But just imagine, you will have to open you documents, browse to folder with this magic file, open it, select text snippet, copy, return to browser and finally paste it. I guess, you can find better way to spend this time, because with Pastemark you will have to do it with at most three clicks — in three seconds!

Ok, but I have nice desktop application which can do this job.

Again, you can keep using it, if you like it. But Pastemark has advantages even in this case:

  1. It's fully cross-platform. If you can run your browser and your browser can run JavaScript, you can run Pastemark! No matter if your use Windows, Linux, Mac, FreeBSD, Android, iOS etc.
  2. It's easy to stay in sync. Even if you have a bunch of devices with different OS and browsers, you can easily sync your snippets between them using such services like Xmarks or similar (Personally I use Xmarks).

What Pastemark is?

Pastemark is a simple tool for keeping text snippets in browser and paste them quickly into fields.

Do I need it?

Yes, if you have to type same texts every day and want to optimize this process.

How can I use it?

Just fill fields on the left, press "Generate!" and drag generated bookmarklet onto your bookmark toolbar.

Creating Pastemarks is easy!

There are two ways to create pastemark:

  1. Pastemark Fast Tool
  2. HTML-guru

I gonna describre both of them and feel free to choose which one is better for you.

Pastemark Fast Tool

Pastemark Fast Tool is a simple bookmarklet which allows automatically configure Pastemark to wanted field and text. Step-by-step:

  1. Drag and drop this link to your bookmarks toolbar: Pastemark Fast!
  2. Go to page where you want to use Pastemark.
  3. Click on «Pastemark Fast!» bookmarklet and then on target field (where your text should be pasted).
  4. New pastemark page will be opened with options already set to work with your field.
  5. Check and correct (if needed) options and press «Generate!» button.
  6. Drag and drop generated bookamrklet onto bookmarks toolbar.
  7. You've done!

HTML-guru

If you are familiar with HTML it may be easier for you to configure Pastemark by yourself.

  1. Go to page where you want to use Pastemark.
  2. Identify target field name or id using your favorite tool (Firebug is awesome) or just look into page source.
  3. Open Pastemark site.
  4. Type bookmarklet title.
  5. Select how bookmarklet should identify target field (by id, field name or your click; however, id is the best way).
  6. If you have chosen id or name, type target id or name (respectivly) into «Target» field.
  7. Type text you want to be pasted.
  8. You've almost done! Press «Generate!» and drag and drop bookmarklet onto your bookmarklet toolbar.
  9. You've done!
